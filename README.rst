Stencil-flask
=============

Stencil-flask is `Stencil<https://github.com/yumike/Stencil>`_
powered template for simple flask app with
some batteries like ``alembic`` and ``tests`` included.

Using
-----

1. ``mkproject appname``
2. ``pip install stencil-flask``
3. ``stencil flask .``
